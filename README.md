## Kennesaw Motorsports DBC library

This is generated code for the Kennesaw Motorsports DBC repository. It is generated using the `dbc-codegen` crate, which is a code generator for DBC files.

### Install

Run the following Cargo command in your project directory:
```shell
cargo add ksu-rs-dbc --git "https://github.com/KSU-MS/ksu-ms-dbc" --branch rust
```

Or add the following line to your Cargo.toml:
```toml
[dependencies]
ksu-rs-dbc = { git = "https://github.com/KSU-MS/ksu-ms-dbc", branch = "rust", version = "0.1.0" }
```
